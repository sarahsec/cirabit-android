### Added
- Emoji reactions for messages (long-press picker and grouped counters).
- New mesh reaction protocol packet (`MSG_REACTION`) with encrypted private reactions.
- Automatic app version checks every 12 hours.
- A one-time update available dialog per release.
- Update check toggle and status in About settings.

### Changed
- Tapping update settings now opens the official download page:
  https://cirabit.smaia.dev/download
- Updated deprecated Android APIs for better compatibility with newer SDKs:
  - Foreground service stop behavior
  - BLE characteristic notify/write calls
  - Compose clipboard/back-icon/tab-indicator usage

### Fixed
- Improved update-check reliability with fallback endpoint support.
- Prevented false update-check errors when UI coroutines are cancelled.
- Replaced deprecated one-shot location request flow and removed unchecked cast warnings in runtime parsers.
